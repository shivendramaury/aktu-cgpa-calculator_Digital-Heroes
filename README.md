# AKTU CGPA Calculator

A responsive and user-friendly AKTU SGPA/CGPA Calculator built using HTML, CSS, and JavaScript.

## Features

- Calculate SGPA for each semester
- Calculate percentage automatically
- Dynamic subject loading based on year
- Dark mode support
- Download result as PDF
- Mobile responsive design
- Reset all data functionality
- Ignores 0-credit subjects (Constitution of India, Indian Tradition) during SGPA and percentage calculation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- html2canvas
- jsPDF

Clone the repository:

```bash

```

Open the project folder:

```bash
cd aktu-cgpa-calculator
```

Open `index.html` in your browser.
